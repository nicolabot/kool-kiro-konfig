# Backend Developer Role

This config provides a tailored AI assistant for backend developers on the AnyCompany team.

## What You Get

- **Agent**: Assists with Lambda function development, Spring Cloud Function patterns, Comprehend/Bedrock integration, and error handling
- **Context**: Guidelines for Spring Cloud Function beans, Maven multi-module conventions, SLF4J structured logging, AWS client configuration, PII redaction patterns
- **Skill**: `lambda-scaffolding` — auto-activates when you ask to create a new Lambda function

## Tools

- `read` — read project files
- `write` — create and edit files
