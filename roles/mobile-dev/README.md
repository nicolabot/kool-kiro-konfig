# Mobile Developer (iOS) Role

This config provides a tailored AI assistant for iOS developers on the AnyCompany team.

## What You Get

- **Agent**: Assists with Swift/SwiftUI development, Human Interface Guidelines compliance, navigation patterns, and API integration
- **Context**: Guidelines for SwiftUI conventions, HIG, MVVM architecture, local storage, and AnyCompany backend integration
- **Skill**: `screen-scaffolding` — auto-activates when you ask to create a new screen

## Tools

- `read` — read project files
- `write` — create and edit files
