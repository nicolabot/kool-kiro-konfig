# DevOps Engineer Role

This config provides a tailored AI assistant for DevOps engineers on the AnyCompany team.

## What You Get

- **Agent**: Assists with CloudFormation template authoring, VPC/networking design, CI/CD pipeline setup, monitoring, and security
- **Context**: Guidelines for nested stack conventions, VPC endpoint patterns, KMS/IAM least-privilege, CloudWatch alarms, deployment pipeline
- **Skill**: `cfn-template-scaffolding` — auto-activates when you ask to create a new CloudFormation template

## Tools

- `read` — read project files
- `write` — create and edit files
