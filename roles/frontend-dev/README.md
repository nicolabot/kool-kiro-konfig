# Frontend Developer Role

This config provides a tailored AI assistant for frontend developers on the AnyCompany team.

## What You Get

- **Agent**: Assists with UI component development, design system, accessibility, state management, and API integration
- **Context**: Guidelines for component patterns, CSS/styling, testing standards, and REST/GraphQL integration
- **Skill**: `component-scaffolding` — auto-activates when you ask to create a new component

## Tools

- `read` — read project files
- `write` — create and edit files
