# Team Roles

## Frontend Developer (`frontend-dev`)

Assists with UI component development, design system adherence, accessibility compliance, and state management patterns.

**Config includes**:
- Agent with `read` and `write` tools
- Guidelines for component patterns, CSS/styling, testing, API integration
- Skill: `component-scaffolding` — scaffold new UI components with tests and styles

## Backend Developer (`backend-dev`)

Assists with Lambda function development using Spring Cloud Function, Amazon Comprehend/Bedrock integration, API design, and error handling.

**Config includes**:
- Agent with `read` and `write` tools
- Guidelines for Spring Cloud Function Lambda patterns, Maven multi-module, structured logging, AWS client beans
- Skill: `lambda-scaffolding` — scaffold new Lambda modules with Spring Cloud Function bean and CloudFormation resource

## Mobile Developer — iOS (`mobile-dev`)

Assists with iOS development using Swift/SwiftUI, Human Interface Guidelines compliance, navigation patterns, and API integration with the AnyCompany backend.

**Config includes**:
- Agent with `read` and `write` tools
- Guidelines for Swift/SwiftUI conventions, HIG, navigation, local storage, API integration
- Skill: `screen-scaffolding` — scaffold new iOS screens with SwiftUI view, view model, and preview

## DevOps Engineer (`devops-eng`)

Assists with CloudFormation template authoring, VPC/networking design, CI/CD pipeline setup, monitoring configuration, and security best practices.

**Config includes**:
- Agent with `read` and `write` tools
- Guidelines for CloudFormation nested stacks, VPC endpoints, KMS/IAM, CloudWatch, deployment pipeline
- Skill: `cfn-template-scaffolding` — scaffold new CloudFormation child templates integrated into main.yaml
